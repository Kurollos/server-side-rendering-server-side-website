## Beschrijving
Voor dit project ontwikkel ik een server-side website voor een opdrachtgever. De website haalt content dynamisch op uit een database en een REST API. Het ontwerp volgt de Content First methode, waarbij de inhoud centraal staat en leidend is voor de structuur en vormgeving van de website.

De website wordt opgebouwd met behulp van Liquid templates, zodat pagina’s server-side gerenderd worden. Dit zorgt voor betere prestaties, overzichtelijke code en dynamische contentweergave.

## Gebruik
De website is bedoeld als dynamisch platform waarbij:

Content automatisch wordt geladen vanuit een database en API
Pagina’s server-side worden gegenereerd
De structuur duidelijk en schaalbaar blijft

Gebruikers kunnen de website bezoeken via de browser zonder dat client-side rendering nodig is voor basisfunctionaliteit.

## Kenmerken
Het ontwerp van de website volgt de Content First methode, waarbij een sitemap wordt gebruikt als basisstructuur. Op basis hiervan worden wireframes gemaakt om de layout en gebruikersflow te bepalen, en HTML-prototypes om zowel de visuele als functionele aspecten te valideren.

De website maakt gebruik van server-side rendering met Liquid templates, wat zorgt voor snelle laadtijden en betere SEO-prestaties. Daarnaast wordt er gewerkt met dynamische content die wordt opgehaald uit een database en via een REST API wordt geïntegreerd.

Tot slot heeft de website een overzichtelijke structuur, met herbruikbare componenten en een logische opbouw van pagina’s.
